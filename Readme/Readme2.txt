1) Минимизация сткрипта - это удаление из кода всех несущественных символов с целью
    уменьшения объёма файла скрипта и ускорения его загрузки.
    Программа JSMin, YOI Compressor

2) Обфускация -  альтернативный способ сокращения кода. Так же вырезает пробелы, коментарии
    и ко всему прочему изменяет созданые пользователем имена переменных и функция
    уменьшая их до минимального числа букв. Иногда требуеться настроить этот процесс что бы 
    случайным образом не изменились названия стороних библиотек.
    Программа Dojo Compressor(Shrink Safe)
    
3) Агрегация - это объединение нескольких файло в один. Иногда склейка файлов может давать 
    сбой из-за некорректных коментариев в конце файлов.

4) HTTP-сжатие - это способ сжать контент(в основном текстовый), который передаётся
    с веб серверов через интернет в браузеры. Уменьшается кол-во байт влияя на производительность.
    Этот метод доставки сжатого контента основан на стандартах HTTP/1.1 все современные
    браузеры поддерживают данный протокол.
    Нужно на стороне сервера задавать сжатие файлов.

Повышение производительности в коде так же нужно 
    минимизаровать запросы Ajax
    правильное использование переменных
    правильная работа с DOM
    изменение производительности


Конвертация - это изменение кода одного языка программироавния в другой код языка
Транспиляция - изменение структуры кода одного языка программирования


О СЕТИ

Каждое устройство имеет IP адресс, каждый сайт имееет IP адресс сервера
на котором они лежат. Но чтобы не возникало путаницы с сайтами которые лежат 
на одном сервере у сайтом есть имена которые и дают понять как к нему обратиться
на даном сервре (перейдя по IP сервера). 
Domain Name - это и есть название сайта которое ищеться на сервере. 
Принцип телефоной книги, есть имена и есть номера, искать человека по номеру совсем не удобно
но искать номер по имени намного удобней.
DNS - Domain Name Server - сервер где храняться домены
Клиент через браузер посылает домен на DNS сервер и браузер получает ответ в виде IP сервера
где храняться данные сайта. Браузер обращается к этому серверу по IP и 

Host это и есть сервер. 
Что такое ПАКЕТ - это даные которые разбиваються на 0 и 1 деляться на части которые именуються ПАКЕТ
и отправляються по оптоволокну или по воздуху, потом собираються воедино 

ХАБ это какая-то приблуда типо Switch но только вместо того чтобы передать ответ одному
клиенту на на один запрос, он делает на один запрос один и тот же ответ на много абонентов

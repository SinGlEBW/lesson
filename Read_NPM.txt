NPM Node.js Package Manager- это база хранящяя кучу пакетов, как рабочих так и всякий мусор. Любят её кнечно за качественный
инструмент который можно установить и пользоваться.

Команды для работы с NPM.. Устанавливаеться это добро или локально или глобально

npm install название пакета //  ./node_modules
npm install -g название пакета // глобальный вариант путь такой <prefix>/lib/node_modules - <prefix>  зависит

npm install название пакета @ указываеться конкретная версия пакета

npm uninstall название пакета // удалить
npm update название пакета //установить

npm хранит копию пакета в кеш на случай если захочеться установить заново после удаления
npm cache clean //удаляет кеш 

####---о package.json---###
устанавливаеться npm init.

кстате говоря пакеты успешно могут быть установлены без package.json, но его ставить полюбому нужно.
Разделы в этом файле.

dependencies - отвечает за пакеты которые требуються для непосредственной работы проекта

devDependencies- эта ветка отвечает за пакеты которые требуются для разработки проекта  

package.json удаобная штука, хранит все названия установленых пакетов. При переносе этого файла можно установить
с лёгкостью все те пакеты которыми мы пользовались прописав 
npm install
Обычно перед передачей проекта папку node_modules удаляют оставив файл package.json в дальнейшем просто прописываеться
команда npm install и всё встаёт на свои места

npm install --save-dev название пакета //или сокращёный -d сохранаяет пакет в разделе devDependencies.

####---о gulp---###
как обычно проходит установка пакетов
npm install gulp-cli -g установка cli означает что команды установки будут использоваться в консоли
npm install gulp -D далее непосредственно установка и сохранение в package.json в разделе devDependencies посредством --save-dev

так же после -D могут перечисляться модули для установки
npx -p touch nodetouch gulpfile.js дополнительный файл. 
 
 
 gulp - это сборщик для небольших проектов. Существует их много на слуху есть WEBPACK это для больших проектов

 bundler - тоже собирет проекта
 Taskrunner - выполняет проекты(таски - задачи)
